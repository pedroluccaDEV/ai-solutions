# 💬 AI Multi-Agent Chat Widget para Websites

## 📌 O que é

Um widget de chat com inteligência artificial que pode ser integrado a qualquer site via JavaScript, permitindo a criação de um sistema de atendimento automatizado baseado em múltiplos agentes especializados.

Cada agente pode assumir um papel específico dentro da conversa, atuando de forma coordenada e compartilhando contexto.

---

## 🎯 O que faz

O widget fornece um ambiente de atendimento inteligente capaz de:

* Responder usuários em tempo real
* Direcionar interações para agentes especializados
* Executar diferentes tipos de atendimento dentro do mesmo chat
* Utilizar contexto e histórico para respostas mais precisas

---

## 🧠 Sistema Multi-Agentes

O chat suporta múltiplos agentes atuando dentro da mesma conversa.

### Características:

* Cada agente possui uma função específica (ex: suporte, vendas, FAQ)
* Ativação baseada em contexto ou triggers definidos
* Possibilidade de segmentação por intenção do usuário
* Troca dinâmica de agentes durante a conversa

---

## 🔁 Memória Compartilhada

Todos os agentes operam sobre uma memória compartilhada, permitindo:

* Continuidade da conversa mesmo com troca de agentes
* Acesso ao histórico completo de interações
* Respostas mais coerentes e contextualizadas
* Colaboração entre agentes

---

## 🧠 Configuração dos Agentes

Os agentes podem ser configurados sem necessidade de código.

### Personalização inclui:

* Definição de persona (tom e comportamento)
* Instruções específicas (prompt base)
* Regras de atuação
* Definição de triggers (tags, intenções ou palavras-chave)

---

## 📚 Gestão de Conhecimento

O sistema permite alimentar os agentes com diferentes fontes de informação:

* Conteúdos institucionais
* Informações sobre produtos e serviços
* FAQs
* Dados operacionais

### Objetivo:

Garantir que os agentes respondam com base em informações reais e relevantes do negócio.

---

## 🎓 Treinamento

Os agentes são treinados a partir de:

* Instruções definidas pelo usuário
* Conteúdo inserido na base de conhecimento
* Contexto acumulado das interações

O treinamento é orientado a maximizar a precisão das respostas e reduzir dependência de inputs repetitivos.

---

## ⚙️ Como funciona

### Fluxo:

1. Um ou mais agentes são configurados
2. Conhecimento e instruções são definidos
3. Triggers de ativação são atribuídos aos agentes
4. Um script JavaScript é gerado
5. O script é inserido no site
6. O widget é carregado automaticamente
7. Usuários interagem com o chat
8. O sistema seleciona e ativa o agente mais adequado
9. A resposta é gerada com base em contexto e memória

---

## 🔌 Integração

A integração é feita através de um script JavaScript simples, compatível com qualquer site.

* Implementação rápida (copy/paste)
* Sem necessidade de backend próprio
* Comunicação via webhook
